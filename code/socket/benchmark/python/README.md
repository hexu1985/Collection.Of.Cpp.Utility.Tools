### Python Socket编程示例

#### streamer.py

原始代码: PYTHON网络编程 第3版 5.2章

启动服务器端
```
$ python3 streamer.py -p 9999 "127.0.0.1"
```

启动客户端
```
$ python3 streamer.py -c -p 9999 "127.0.0.1"
```
