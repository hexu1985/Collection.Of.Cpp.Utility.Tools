导读

在 YAML 出现之前，XML 是主流的配置文件格式。XML 文件中使用大量的标签和属性，并且结构层次复杂、嵌套深度大，给使用者造成了一定的困难。为了解决 XML 的种种问题，Clark Evans、Ingy döt Net 和 Oren Ben-Kiki 等人在 2001 年开始设计一种新的数据序列化语言，并提出了以下设计目标：

  • 简洁：使用更少的符号和更简单的语法规则，使文件更加简洁
  • 易读：采用缩进和换行等排版方式，使数据结构更加清晰易读
  • 易写：语法规则简单易懂，方便手动编写和修改
  • 可扩展：支持自定义数据类型和标签，满足不同场景的需求

基于这些设计目标，YAML 应运而生。目前，YAML 广泛应用在 DevOps，云计算和容器技术等领域。

最初，YMAL 设计成与 XML 类似的标记语言，其全称是 Yet Another Markup Language，意为“另一种标记语言”。现在，YAML 的全称是 YAML Ain't Markup Language，意为“YAML 不是一种标记语言”。这也体现了YAML 最初的设计理念，即关注数据序列化，而非传统的标记。

有趣的是，YAML 是一个递归缩写，即缩写本身包含了自己的名字。

YAML文件的扩展名通常是.yaml，其设计灵感来源于 JSON、XML 和 Python 的缩进风格，具有更加人性化的数据格式。YAML 文件常用于配置文件和数据交换。

基本语法

1. 键值对 (Key-Value Pair)

  • YAML 使用 key: value 的形式表示键值对，冒号后面必须有一个空格
name: Xiao Ming
age: 18
  • C++ 示例
std::string name = config["name"].as<std::string>();
int age = config["age"].as<int>();

2. 列表 (List)

  • 使用 - 表示列表项，每一项占一行
fruits:
  - Apple
  - Banana
  - Orange
  • C++ 示例
YAML::Node fruits = config["fruits"];
for (const auto& fruit : fruits) {
    std::cout << "- " << fruit << std::endl;
}

3. 嵌套结构

  • YAML 支持嵌套的键值对和列表
person:
  name: Li Hua
  age: 25
  hobbies:
    - Reading
    - Swimming
  • C++ 示例
YAML::Node person = config["person"];
std::string p_name = config["person"]["name"].as<std::string>();
int p_age = config["person"]["age"].as<int>();

YAML::Node hobbies = config["person"]["hobbies"];
for (const auto& hobby : hobbies) {
    std::cout << "        - " << hobby << std::endl;
}

4. 多行字符串

  • 使用 | 保留换行符，或使用 > 折叠换行符
description: |
  This is a multi-line
  string in YAML.
summary: >
  This is a folded
  multi-line string.
  • C++ 示例
std::string description = config["description"].as<std::string>();
std::string summary = config["summary"].as<std::string>();

5. 注释

  • 使用 # 表示注释
# This is a comment
name: Bob

6. 数据类型

  • YAML 自动推断数据类型，如字符串、整数、浮点数、布尔值
int_v: 42
float_v: 3.14
bool_v: true
string_v: "Hello, YAML!"
  • C++ 示例
std::cout << "int: " << config["int_v"].as<int>() << std::endl;
std::cout << "float: " << config["float_v"].as<float>() << std::endl;
std::cout << "bool: " << config["bool_v"].as<bool>() << std::endl;
std::cout << "string: " << config["string_v"].as<std::string>() << std::endl;

C++ 完整示例

在 C++ 中，通常需要使用第三方库 yaml-cpp 来解析和生成 YAML 文件。

1. Linux 安装 yaml-cpp

sudo apt update
sudo apt install libyaml-cpp-dev

检查 yaml-cpp 三方库是否安装成功

dpkg -l | grep yaml-cpp

附带卸载 yaml-cpp 命令：

sudo apt remove libyaml-cpp-dev

2. C++ 代码示例

  • 演示上述六种 YAML 语法 (读取 YAML 文件)

2.1 先来阅览一下整个 C++ 工程目录

2.1 首先，准备一个 YAML 文件，config.yaml 内容如下：

# This is a YAML example
name: Xiao Ming
age: 18

fruits:
    - Apple
    - Banana
    - Orange

person:
    name: Li Hua
    age: 25
    hobbies:
        - Reading
        - Swimming
          
description: |
    This is a multi-line
    string in YAML.
summary:
    This is a folded
    multi-line string.
    
integar: 42
float: 3.14
boolean: true
string: "Hello, YAML!"

2.2 其次，编写 yaml.cpp 源文件

#include <iostream>
#include <yaml-cpp/yaml.h>  /* yaml-cpp header file */

int main() {
    /* load yaml file */
    YAML::Node config = YAML::LoadFile("/home/cui/cui/yaml/config.yaml");

    /* key-value */
    std::string name = config["name"].as<std::string>();
    int age = config["age"].as<int>();
    std::cout << "Name: " << name << std::endl;
    std::cout << "Age: " << age << std::endl << std::endl;

    /* list */
    YAML::Node fruits = config["fruits"];
    std::cout << "fruits: " << std::endl;
    for (const auto& fruit : fruits) {
        std::cout << "- " << fruit << std::endl;
    }
    std::cout << std::endl;

    /* nested structure */
    YAML::Node person = config["person"];
    std::cout << "person info:" << std::endl;
    std::string p_name = config["person"]["name"].as<std::string>();
    int p_age = config["person"]["age"].as<int>();
    std::cout << "    name: " << p_name << std::endl;
    std::cout << "    age: " << p_age << std::endl;

    YAML::Node hobbies = config["person"]["hobbies"];
    std::cout << "    hobbies:" << std::endl;
    for (const auto& hobby : hobbies) {
        std::cout << "        - " << hobby << std::endl;
    }
    std::cout << std::endl;

    /* multi-line string */
    std::string description = config["description"].as<std::string>();
    std::string summary = config["summary"].as<std::string>();
    std::cout << description << std::endl;
    std::cout << summary << std::endl << std::endl;

    /* data type */
    std::cout << "int: " << config["int_v"].as<int>() << std::endl;
    std::cout << "float: " << config["float_v"].as<float>() << std::endl;
    std::cout << "bool: " << config["bool_v"].as<bool>() << std::endl;
    std::cout << "string: " << config["string_v"].as<std::string>() << std::endl;

    return 0;
}

2.3 然后，编写 CMakeLists.txt 文件

cmake_minimum_required(VERSION 3.16)
project(yaml_demo)

find_package(yaml-cpp REQUIRED)

add_executable(${PROJECT_NAME} yaml.cpp)

# 链接 yaml-cpp 库
target_link_libraries(${PROJECT_NAME}
    yaml-cpp
)

2.4 编译生成可执行程序

cd build
cmake ../
make

2.5 运行可执行程序

通过本文可以 YAML 快速入门，了解基本用法和 C++ 编译过程。
